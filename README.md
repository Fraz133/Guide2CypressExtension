# Guide2Cypress

This VS Code extension helps you learn Cypress by providing examples, snippets, and guides.

## Features
- Snippets for common Cypress commands
- Quick links to Cypress docs
- Run Cypress tests from VS Code

## Requirements
Node.js ≥ 18 and Cypress installed globally.

## Extension Settings
None for now.
